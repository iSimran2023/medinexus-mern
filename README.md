# MediNexus: Advanced Patient Scheduling & Clinical Management Suite

A modern MERN stack application designed to streamline the doctor-patient appointment process. Built with TypeScript, React, and Node.js.

## Features
- **Multi-Role Authentication:** Separate portals for Patients, Doctors, and Administrators.
- **Dynamic Scheduling:** Doctors can manage their session availability.
- **Intelligent Booking:** Patients can search for doctors by specialty and book available slots.
- **Admin Dashboard:** Comprehensive control over user accounts, specialties, and session logs.
- **Cloud-Native Architecture:** Scalable backend with JWT-secured RESTful APIs.

## Installation

### Prerequisites
- Node.js & npm
- MongoDB

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   # Server
   cd server && npm install
   # Client
   cd client && npm install
   ```
3. Create a `.env` file in the `server` directory (refer to `.env.example`).
4. Start the development servers:
   ```bash
   # Server
   npm run dev
   # Client
   npm run dev
   ```
